This project is a console based application which is written in C programming language.<br>

**Features**<br>
* This is an health recommendation system which tells tou about your diet workout and various other things based on you requirement.
* And we can calculate calories for a persion in a day and BMI of a persion based on height and weight.
* In this application we can register and login.
